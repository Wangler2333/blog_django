# blog_django

### 模块

- 使用 Amaze UI 作为前端界面
- 使用 Editor.md 作为 Markdown 编辑器

- **感谢Amaze UI和Editor.md开源贡献**

---

- 使用 Django 作为后端框架
- 使用 MySQL 作为数据库
- 使用 Redis 作为session数据库和缓存数据库
- 使用 celery 作为分布式异步任务处理器
- 计划使用 FastFDS 作为分布式文件存储器


### 已实现效果

- 实现注册博客
- 实现markdown在线编辑
- 实现后台markdown转换为html标签
- 实现前端渲染转换后的html标签
- 实现注册用户动态个人主页


### 计划实现效果
- 实现评论
- 实现动态个人相册
- 实现私信功能
- 实现多级权限
